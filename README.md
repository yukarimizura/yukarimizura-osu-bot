# osu! Discord Bot (Python)

A Discord bot for osu! featuring: (for now)

- 🎮 Recent plays
- 🏆 Top plays
- 📊 Trace / Score history
- 📈 PP recalculation
- 🖼️ Beatmap embeds
- 📄 Pagination

## Installation

```bash
git clone https://github.com/yourname/osu-discord-bot.git
cd osu-discord-bot

pip install -r requirements.txt
```

Create a `.env` file:

```env
DISCORD_TOKEN=YOUR_TOKEN
OSU_CLIENT_ID=YOUR_CLIENT_ID
OSU_CLIENT_SECRET=YOUR_CLIENT_SECRET
```

Run:

```bash
python bot.py
```
